#Compositor

Window Manager/Compositor for Wayland.

- Supports Fullscreen Games/Displays Tested with Quake/Command and Conquer Generals.
- Supports GNUstep Applications as first-class applications.
- Supports Xwayland.
- Supports Client Side Deocrations by default. Server Side decorations are an option.
- Configured via SystemPreferences.app Module which integrates into GNUstep SystemPreferences.app
- Supports VR Display Extensions and VR Isolation using DRM-Lease.
